# Global Inflation Data Analysis Project

## 📌 Project Overview
This repository contains an end-to-end data analysis project focusing on global economic inflation trends. Starting from fragmented raw economic files, the data was thoroughly cleaned, structured into a main working dataset, and analyzed to answer critical macroeconomic questions and extract actionable financial insights.

The entire workflow—from data quality engineering to business intelligence reporting—was built and executed entirely within **Microsoft Excel**.

---

## 📂 Repository Structure
The data in this project is organized into structured sheets representing the lifecycle of the analysis:
* **Raw Data**: Unprocessed economic datasets containing inconsistencies and missing values.
* **Main Data**: The finalized, clean, and structured master table used for advanced querying.
* **Question 1 to 5 Sheets**: Dedicated workbenches isolating complex mathematical and financial queries.
* **Insights**: The final strategic layer summarizing global trends, systemic volatility, and macroeconomic impacts.

---

## 🛠️ Key Technical Implementations (Excel Capabilities)

### 1. Data Cleaning & Transformation
* **Handling Anomalies**: Audited the `Raw Data` to identify missing records, format inconsistencies, and null blocks.
* **Standardization**: Converted values into clean data types (Dates, Numeric, Percentages) to form a robust `Main Data` set.

### 2. Advanced Formula Engineering & Data Lookup
* Utilized advanced lookup capabilities (`XLOOKUP` / `VLOOKUP` / `INDEX-MATCH`) to map economic metrics across multiple dimensions.
* Applied logical and statistical functions (`IF`, `IFS`, `AVERAGEIFS`, `SUMIFS`, `COUNTIFS`) to isolate specific economic parameters.

### 3. Business Intelligence & Macroeconomic Analysis
* **Trend Tracking**: Built comparative analysis sheets to study inflation spikes and economic cooling periods across different decades/regions.
* **Data Aggregation**: Summarized massive datasets using **Pivot Tables** and conditional aggregations to answer explicit analytical questions (Sheets 1–5).
* **Insight Synthesis**: Translated complex statistical trends into high-level executive summaries on inflation drivers and market behaviors.

---

## 📈 Key Metrics & Analytical Framework
The project explicitly solves business-critical data questions including:
* Identifying historically high inflation eras and correlating factors.
* Measuring year-over-year baseline shifts in consumer index data.
* Isolating regional volatility to highlight stable vs. high-risk economic zones.
